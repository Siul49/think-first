# Thinking Mode — 사고 강제 에이전트 전환

## 목표

skill-pack을 "AI가 대신 해주는 도구"에서 **"사용자에게 사고를 강제하는 도구"**로 전환한다.
모든 작업에 **질문 → 결정 → 실행 → 회고** 사이클을 적용한다.

## 성공 기준

1. 어떤 스킬이든 실행 전 최소 1개 소크라테스 질문이 발생한다
2. 트레이드오프가 있는 선택지에서 사용자가 근거와 함께 답하지 않으면 진행하지 않는다
3. 모든 작업 완료 후 회고가 `.agent/reflections/YYYY-MM-DD.md`에 기록된다
4. 사용자가 "그냥 해줘"라고 해도 최소 1개 질문에는 답해야 진행된다
5. 배포 가능한 상태를 유지한다 (다른 사용자도 설치 가능)

## 범위

### In

- 사고 사이클(Thinking Cycle) 공유 리소스 신규 생성
- 모든 도메인 스킬(14개)에 사고 사이클 적용
- 회고 시스템 (`.agent/reflections/`) 신규 구축
- antigravity.md 핵심 철학 변경
- clarification-protocol.md 강화 (스킵 불가 정책)
- install.sh에 reflections 디렉토리 포함

### Out

- 검증 스킬(verify-*) 변경 — 이미 검증 목적이므로 사고 사이클 불필요
- 서브에이전트 동작 변경 — 자동화 유지 (사용자 대면이 아닌 백그라운드)
- Hook 스크립트 변경 — 시스템 자동화 유지
